**# Nairobi Expressway Cost-Benefit Analysis (CBA)**

**## Project Overview**

This project evaluates the economic viability of the Nairobi Expressway Public-Private Partnership (PPP) by applying Cost-Benefit Analysis (CBA) techniques using 

Python. The analysis quantifies financial performance, operational efficiency, and socioeconomic impacts through data visualization and key performance indicators.

**## Objectives**

- Evaluate project profitability

- Analyze annual revenues and operating costs
  
- Calculate Net Present Value (NPV)
  
- Estimate Benefit-Cost Ratio (BCR)
  
- Assess Return on Investment (ROI)
  
- Visualize financial and operational performance
  
- Provide evidence-based policy recommendations

**## Tools Used**

- Python
  
- Pandas
  
- NumPy
  
- Matplotlib
  
- Seaborn
  
- Jupyter Notebook
  
- Power BI

**## Key Metrics**

- Net Present Value (NPV)
  
- Benefit-Cost Ratio (BCR)
  
- Internal Rate of Return (IRR)
  
- Return on Investment (ROI)
  
- Payback Period
  
- Annual Traffic Volume
  
- Toll Revenue
